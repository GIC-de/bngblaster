+-----------------------------------+----------------------------------------------------------------------+
| Command                           | Description                                                          |
+===================================+======================================================================+
| **traffic-start**                 | | Start all traffic (session and streams).                           |
+-----------------------------------+----------------------------------------------------------------------+
| **traffic-stop**                  | | Stop all traffic (session and streams).                            |
+-----------------------------------+----------------------------------------------------------------------+
| **multicast-traffic-start**       | | Start multicast traffic.                                           |
+-----------------------------------+----------------------------------------------------------------------+
| **multicast-traffic-stop**        | | Stop multicast traffic.                                            |
+-----------------------------------+----------------------------------------------------------------------+
